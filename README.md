# BFInterpreter

Простой интерпретатор для языка Brainfuck

## Пример использования

`hello-world.bf`:
```bf
>+++++++++[<++++++++>-]<.>+++++++[<++++>-]<+.+++++++..+++.>>>++++++++[<++++>-]<.>>>++++++++++[<+++++++++>-]<---.<<<<.+++.------.--------.>>+.>++++++++++.
```
Запуск
```bash
dotnet run hello-world.bf
```
